
冒泡排序
    相邻2个数进行比较，一趟下来后最大值在最右边，下一次则从第一位开始到倒数第二位，以此类推。

选择排序
    第一个数与其他数进行比较，一趟下来后最小值在最左边，下一次则从第二位开始到倒数第一位，以此类推。

双向选择排序
    先第一个数与最后一个数比较，保证第一个数小，最后一位数大；然后第一个值其他数进行比较，最后一个数与其他数进行比较。

插入排序
    将未排序的新元素（从第2个开始），跟已经排序的元素进行比较（从后往前），如果新元素小于已排序的元素右移，插入新元素


双向选择排序 比选择排序 减少了一半的外层循环次数；
如果第一趟后是局部有序，则冒泡比较快。
    因为双向选择排序外层循环次数总是length/2,冒泡排序的外层循环次数可能是1次(极端情况:本身已排序）
